rgk-select

rgkSelect is lightweight and simple customization script for select boxes.

Basic initialisation

Just simple select item:

<select id="selectSimple" name="select-simple"> <option>Bangladesh</option> <option>Botswana</option> <option>Canada</option> <option disabled>Dubai (disabled)</option> <option>Ghana</option> <option>Lesotho</option> <option>Malawi</option> </select>

Initialization:

rgkSelect('#selectSimple');

Element selector can be as css selector string or js object. For example:

var mySelect = document.getElementById('#selectSimple'), mySelects = document.querySelectorAll('select'); rgkSelect('form > select.class-name:not(.no-styling)'); // css selector string rgkSelect(mySelect); // Single element object rgkSelect(mySelects); // List of elements objects

Also supports multiple choices:

<select id="selectMulti" name="select-multi" multiple size="3"> <option>Bangladesh</option> <option>Botswana</option> <option>Canada</option> <option disabled>Dubai (disabled)</option> <option>Ghana</option> <option>Lesotho</option> <option>Malawi</option> </select>

Placeholder

If you need add placeholder to your select, just use data-placeholder attribute on one of options (preferably first) for "Single select" and on element itself for "Multiple select":

<select id="selectPlaceholderSingle" name="select-placeholder"> <option data-placeholder selected>Choose country</option> <option>Bangladesh</option> <option>Botswana</option> <option>Canada</option> </select> <select id="selectPlaceholderMulti" name="select-placeholder" multiple data-placeholder="Choose country"> <option>Bangladesh</option> <option>Botswana</option> <option>Canada</option> <option>Ghana</option> <option>Lesotho</option> <option>Malawi</option> </select>

Select all items

If you need option which selects all items in multiple select - just add data-all attribute to select element:

<select id="selectMultiAll" name="select-multi" multiple data-all="Select all items"> <option>Bangladesh</option> <option>Botswana</option> <option>Canada</option> <option>Ghana</option> <option>Lesotho</option> <option>Malawi</option> </select>

Methods

rgkSelect can take aso second argument - method (not required):

init

Initializes the plugin. Can be omitted as called by default.

rgkSelect('#selectSimple', 'init'); the same as rgkSelect('#selectSimple');

build

This method run 'rebuilding' html structure of 'visual select'. Need to use, when structure or 'original select' was changed.

rgkSelect('#selectSimple', 'build');

update

Just update visual select values. Need to use, when 'original select' values was updated dynamically.

rgkSelect('#selectSimple', 'update');

destroy

Remove plugin from element(s).

rgkSelect('#selectSimple', 'destroy');

Events

rgkSelect provides two events attached to original select:

show.rgk.select

Fires, when select dropdown is open

var mySelect = document.getElementById('#selectSimple'); rgkSelect(mySelect); mySelect.addEventListener('show.rgk.select', function() { // dropdown is open });

hide.rgk.select

Fires, when select dropdown is hidden

var mySelect = document.getElementById('#selectSimple'); rgkSelect(mySelect); mySelect.addEventListener('hide.rgk.select', function() { // dropdown is hidden });
